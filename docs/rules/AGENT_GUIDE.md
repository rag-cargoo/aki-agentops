# 🤖 에이전트 가이드 (Agent Guide)

이 문서는 에이전트(Gemini)가 이 워크스페이스에서 효율적으로 작업하기 위한 **고급 기능 및 팁**을 정리한 비밀 매뉴얼입니다.

## 1. 🚀 터보 모드 (Turbo Mode) - 자동 승인 시스템
사용자의 승인 없이 명령어를 연속으로 실행해야 할 때 사용합니다.

### 사용법
1. `.agent/workflows/` 폴더에 `.md` 파일을 생성합니다.
2. 파일 내용 중 `// turbo-all` 주석을 추가합니다.
3. 사용자에게 **"OOO 워크플로우 실행해줘"**라고 요청합니다.

### 에이전트 지침
*   반복적인 작업(초기 세팅, 빌드/배포)은 반드시 워크플로우로 정의하여 사용하십시오.
*   단, **파괴적인 명령어(삭제 등)**가 포함된 경우 터보 모드 사용에 신중해야 합니다.

## 2. 📂 멀티 프로젝트 내비게이션
이 워크스페이스는 여러 개의 프로젝트가 공존합니다.

*   **길을 잃었을 때**: 무조건 루트의 `task.md`로 돌아가십시오.
*   **새 프로젝트 시작**: `projects/` 폴더에 디렉토리를 만들고, 반드시 `prj-docs/` 구조(task, TODO, ROADMAP)를 갖춰야 합니다.

## 3. 📝 문서 관리 원칙
*   **기억보다 기록**: 중요한 내용은 대화(Memory)에 의존하지 말고 반드시 파일(Markdown)에 기록하십시오.
*   **단기/장기 구분**:
    *   버그, 잡일 -> `TODO.md`
    *   기능 추가 -> `ROADMAP.md`

이 가이드를 준수하여 **빠르고 프라이빗한** 개발 환경을 유지하십시오.
