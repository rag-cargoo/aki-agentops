# 🔧 전체 트러블슈팅 인덱스 (Troubleshooting Index)

이 파일은 워크스페이스 내 모든 프로젝트에서 발생한 문제와 해결책을 모아둡니다.
같은 문제가 재발하거나 다른 프로젝트에서 유사한 문제가 발생했을 때 참고하십시오.

---

## 📝 사용 방법

사용자가 명시적으로 "이것을 트러블슈팅 문서로 작성해줘"라고 요청할 때만 문서를 작성합니다.

### 문서 작성 프로세스
1. 사용자가 특정 문제 해결을 트러블슈팅 문서로 요청
2. 해당 프로젝트의 `prj-docs/troubleshooting/` 폴더에 `.md` 파일 생성
3. 이 인덱스 파일에 링크 추가

**문서 형식**:
- 문제 증상
- 원인
- 해결 방법
- 예방 방법 (선택)
