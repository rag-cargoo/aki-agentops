# 🌐 GitHub Pages Documentation Expert

이 스킬은 **GitHub Pages**와 **Docsify**를 활용하여 워크스페이스 내의 모든 지식을 고품질 웹 문서로 변환하고 유지보수하는 전문 지침입니다.

## ⚖️ 3대 핵심 미션 (Core Missions)

1.  **배포 무결성 (Zero-Loss Deployment)**: 깃헙 페이지스에 배포되는 모든 지식 문서는 수정 과정에서 단 한 줄의 핵심 정보도 요약되거나 삭제되어서는 안 됩니다.
2.  **YAML형 위계 표준**: Docsify Alert 박스 내부에서 공백을 활용한 깊은 들여쓰기를 적용하여 정보의 계층 구조를 야믈(YAML)처럼 명확히 시각화합니다.
3.  **구조적 정규화**: `H2-선-박스-소제목-선`으로 이어지는 고정 레이아웃을 기계적으로 적용합니다.
