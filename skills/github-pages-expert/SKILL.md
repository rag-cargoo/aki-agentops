# 🌐 GitHub Pages Documentation Expert

이 스킬은 **GitHub Pages**와 **Docsify**를 활용하여 워크스페이스 내의 모든 지식을 고품질 웹 문서로 변환하고 유지보수하는 전문 지침입니다.

## ⚖️ 3대 철칙 (The Iron Rules)

1.  **원자적 섹션 조립 (Atomic Assembly)**: 대규모 리팩토링 시 문서를 한 번에 덮어쓰지 않습니다. 반드시 Step 또는 섹션 단위로 나누어 조립하며, 각 단계마다 물리적 정합성을 확인합니다.
2.  **즉시 시각적 검증 (Visual Proof)**: 파일 수정 직후, 에이전트는 반드시 `cat`, `read_file` 등의 도구를 사용하여 **실제 변경된 내용을 사용자에게 직접 출력하여 증명**해야 합니다.
3.  **무손실 추출 (Lossless Extraction)**: 텍스트 복구 시 에이전트의 메모리보다 `git show` 등 시스템 명령어를 통한 물리적 데이터 추출을 우선하여 요약 및 누락을 원천 차단합니다.
