# 🎨 GitHub Pages Layout Specialist

이 스킬은 **시각적 가독성**을 책임집니다. 물리적 데이터 조립은 하위의 `atomic-doc-skill`에게 명령하고, 본 스킬은 결과물의 **레이아웃과 디자인**만 검수합니다.

## ⚖️ 디자인 철학 (Design Philosophy)

1.  **냉철한 텍스트**: 어떠한 경우에도 이모티콘 사용을 허용하지 않습니다.
2.  **구조적 깊이**: 박스(`> `)와 공백을 활용하여 정보의 깊이를 야믈(YAML) 수준으로 시각화합니다.
3.  **Docsify 최적화**: 다크 테마 코드 블록과 Alert 박스의 정합성을 보장합니다.
