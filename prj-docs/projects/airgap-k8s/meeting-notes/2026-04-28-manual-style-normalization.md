## 안건 1: `02-환경-준비.md`를 매뉴얼 본연의 형식으로 정리할지 결정
- Created At: 2026-04-28 03:20:00
- Updated At: 2026-04-28 03:20:00
- Status: DONE
- 검토사항:
  - 장별 매뉴얼에 운영 규칙, 경로 결정 이유, 문서 역할 설명이 과하게 섞이면 실행 문서로서 가독성이 떨어진다.
  - 제출 매뉴얼은 설명보다 `어떻게 실행하고 무엇을 확인하는지`가 먼저 보여야 한다.
- 결정사항:
  - `manual/02-환경-준비.md`는 실행 절차, 검증 방법, 예상 결과 중심으로 다시 정리한다.
  - 운영 규칙과 문서 역할 구분은 `rules/manual-task-governance.md`에서 관리한다.
