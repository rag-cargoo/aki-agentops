# Manual Download Document Placement

## Summary
- 쿠버네티스 설치용 자산 다운로드 문서는 `manual/01-인터넷-불가능-네트워크-환경-및-리눅스-기본-구성/` 아래에 둔다.
- 이유는 오프라인 자산 다운로드와 반입 준비가 설치 본체(`03`) 이전의 환경 준비 단계이기 때문이다.

## Decision
1. 다운로드 문서는 `manual/01/.../02-쿠버네티스-설치용-자산-다운로드.md`에 둔다.
2. 문서에는 공식문서 링크, 버전 기준, 작업 원본 경로, 반입 번들 경로, 서버 설치 경로를 함께 기록한다.
3. 실제 파일은 `assets/offline-assets/`에 저장하고, 서버 반입용 파일은 `delivery/offline-assets/`에 모은다.

## Impact
- `manual/01`은 환경 재현뿐 아니라 오프라인 자산 준비 절차까지 포함한다.
- `manual/03`은 실제 kubeadm 설치 절차만 유지한다.
- 다음 세션에서는 이 문서를 기준으로 실제 다운로드 목록과 검증 절차를 채운다.
