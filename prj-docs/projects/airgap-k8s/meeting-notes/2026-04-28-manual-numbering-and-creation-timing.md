- Updated At: 2026-04-28 02:05:00

- 결정사항:
  - 프로젝트 루트 `manual/` 아래 매뉴얼 파일은 항상 번호형 한글 파일명을 사용한다.
  - `manual/00-제출-매뉴얼-개요.md`는 공통 서문으로 유지한다.
  - 실제 제출 매뉴얼의 첫 절차 파일은 `02-환경-준비.md`로 시작한다.
  - 실제 장 매뉴얼 파일은 해당 장 작업이 시작될 때 생성한다.
  - 작업 시작 전에는 `manual/`에 장 파일을 미리 만들지 않는다.

## 안건 2: 장별 task 파일 생성 시점
- Created At: 2026-04-28 02:05:00
- Updated At: 2026-04-28 02:05:00
- Status: DONE
- 결정사항:
  - `tasks/*.md`는 매뉴얼과 같은 장 번호를 사용한다.
  - 다만 해당 장의 매뉴얼 파일이 아직 없고 실제 작업도 시작되지 않았다면 task 파일도 만들지 않는다.
  - 착수 전 확정 TODO는 `task.md` 대시보드에만 둔다.
