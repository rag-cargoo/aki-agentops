# Root README and Runbook Separation

## Summary
- 루트 `manual/README.md`는 진입점/index 역할만 유지한다.
- 전체 실행 순서는 별도 문서 `manual/01-전체-실행-순서.md`로 분리한다.

## Decision
1. 파일명만 보고도 바로 알 수 있도록 전체 실행 순서를 별도 파일로 둔다.
2. 루트 `README.md`에는 `전체 실행 순서` 링크와 장별 문서 목록만 둔다.
3. 실제 사용자는 먼저 `01-전체-실행-순서.md`를 보고, 이후 장별 문서로 이동한다.

## Impact
- 루트 문서가 짧아지고 역할이 명확해진다.
- 실행자는 `01-전체-실행-순서.md`만 열어도 전체 흐름을 바로 파악할 수 있다.
