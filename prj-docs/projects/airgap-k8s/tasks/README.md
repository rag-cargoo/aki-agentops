# Task Files (airgap-k8s)

## Purpose
- 이 디렉터리는 제출 매뉴얼 장 순서에 맞춘 sidecar 상세 TODO를 관리한다.
- `task.md`는 대시보드이고, 실제 장별 작업 상태와 세부 근거는 이 디렉터리 파일을 기준으로 본다.

## Rules
- 공통 개요는 `workspace/infra/airgap/kubernetes/airgap-k8s/manual/00-제출-매뉴얼-개요/README.md`를 기준으로 본다.
- 상세 운영 규칙은 `prj-docs/projects/airgap-k8s/rules/manual-task-governance.md`를 권위 소스로 따른다.
- 파일명은 제출 매뉴얼 장 순서와 동일한 번호형 한글 파일을 사용한다.
- 각 파일은 `확정된 TODO`, `완료`, `근거`, `다음 판단 필요사항`만 기록한다.
- 미확정 예상 단계는 회의록이나 매뉴얼 개요에만 남기고 여기에는 올리지 않는다.
