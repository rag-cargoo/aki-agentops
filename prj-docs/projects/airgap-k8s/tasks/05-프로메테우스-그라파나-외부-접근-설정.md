## Sidecar Draft
- 외부 접근 방식과 실제 Service/Ingress 파일이 확정되기 전까지 아래 내용은 제출 매뉴얼에 넣지 않는다.
- 실행 후 접근 URL, 포트, 응답 결과를 매뉴얼에 반영한다.

```bash
kubectl apply -f ops/05-prometheus-grafana-external-access/prometheus/
kubectl apply -f ops/05-prometheus-grafana-external-access/grafana/
kubectl get svc -n monitoring
kubectl get ingress -n monitoring
curl -I http://<PROMETHEUS_ACCESS_HOST>:<PORT>
curl -I http://<GRAFANA_ACCESS_HOST>:<PORT>
```
