# Web Interface Guidelines

Review UI files for compliance with Web Interface Guidelines.

## How It Works
1. Fetch latest guidelines from the source URL below.
2. Read target files (or ask user which files/pattern to review).
3. Check all applicable rules.
4. Output concise findings with actionable file:line guidance.

## Guidelines Source
- `https://raw.githubusercontent.com/vercel-labs/web-interface-guidelines/main/command.md`

Always use the latest source before a review.

## Usage
- Trigger examples:
  - review my UI
  - check accessibility
  - audit design
  - review UX
  - check my site against best practices
