description: Create a demo project to verify Turbo Mode (Auto-Accept)
---

This workflow demonstrates how to use `// turbo-all` to execute commands without user confirmation.

// turbo-all

1. **Create Directory**: make a directory for the demo project.
   - Command: `mkdir -p projects/turbo-demo/prj-docs`

2. **Create Files**: Create the necessary documentation files using touch.
   - Command: `touch projects/turbo-demo/prj-docs/task.md`
   - Command: `touch projects/turbo-demo/prj-docs/TODO.md`
   - Command: `touch projects/turbo-demo/prj-docs/ROADMAP.md`

3. **Verify**: List the created files to confirm success.
   - Command: `ls -R projects/turbo-demo`
